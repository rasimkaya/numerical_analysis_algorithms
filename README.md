## 📖 About the Project
This repository contains Python implementations of fundamental numerical analysis methods used to find the roots of nonlinear equations. The algorithms are applied programmatically, strictly adhering to mathematical principles, and visualized step-by-step using **Matplotlib**. The project focuses on clean code architecture and modular design, following the `DRY` (Don't Repeat Yourself) principle for shared utilities.

## 🔬 Fundamental Theorems (Existence of a Root)

To construct a numerical method, we first need to guarantee the existence of a root within a specific interval. The algorithms in this project are built upon the following fundamental theorems:

### 1. Rolle's Theorem
Let $f$ be a continuous function on the closed interval $[a,b]$ and differentiable on the open interval $(a,b)$. If $f(a) = f(b)$, then there exists at least one $c \in (a,b)$ such that:
$$f'(c) = 0$$

### 2. Mean Value Theorem
Let $f$ be continuous on $[a,b]$ and differentiable on $(a,b)$. Then there exists a $c \in (a,b)$ such that:
$$f'(c) = \frac{f(b) - f(a)}{b - a}$$

### 3. Intermediate Value Theorem
Let $f$ be a continuous function on $[a,b]$. If $K$ is any number between $f(a)$ and $f(b)$, then there exists at least one $c \in (a,b)$ such that:
$$f(c) = K$$

### 4. Bolzano's Theorem
Let $f$ be continuous on $[a,b]$. If the signs of the function values at the endpoints are opposite, meaning:
$$f(a) \cdot f(b) < 0$$
then there exists at least one $c \in (a,b)$ such that $f(c) = 0$. *(This condition guarantees the existence of a root in the interval and forms the core basis of the Bisection ---

## 1️⃣ Bisection Method (Interval Halving)

The Bisection Method is a robust root-finding algorithm that repeatedly bisects an interval and then selects a sub-interval in which a root must lie for further processing. It is a direct application of **Bolzano's Theorem**.

### ⚙️ How it Works:
1. Start with an interval $[a, b]$ such that $f(a) \cdot f(b) < 0$.
2. Calculate the midpoint: $c = \frac{a + b}{2}$.
3. Check the sign of $f(c)$:
   - If $f(a) \cdot f(c) < 0$, the root is in $[a, c]$. Set $b = c$.
   - If $f(b) \cdot f(c) < 0$, the root is in $[c, b]$. Set $a = c$.
4. Repeat until the interval is sufficiently small (based on error tolerance).

### 📈 Convergence and Error:
The maximum error after $n$ iterations is given by:
$$|E_n| \le \frac{b - a}{2^n}$$

## 2️⃣ Fixed-Point Iteration

A point $P$ is called a **fixed point** for a function $g$ if $g(P) = P$. To find a root of $f(x) = 0$, we transform it into the form $x = g(x)$.

### ⚙️ Algorithm:
- **Existence:** If $g \in C[a,b]$ and $g(x) \in [a,b]$ for all $x \in [a,b]$, then $g$ has at least one fixed point in $[a,b]$.
- **Uniqueness:** If $|g'(x)| \le k < 1$ for all $x \in (a,b)$, then the fixed point is unique.

### 🔄 Iteration Steps:
Starting from an initial guess $P_0$, the sequence is generated by:
$$P_n = g(P_{n-1})$$

## 3️⃣ Newton-Raphson Method

The Newton-Raphson method is one of the most powerful and well-known numerical methods for finding roots. It uses the first derivative of the function to find better approximations.

### ⚙️ Derivation:
Using the Taylor series expansion around $P$:
$$P_n = P_{n-1} - \frac{f(P_{n-1})}{f'(P_{n-1})}$$

### 🚀 Implementation Highlights:
In this project, the Newton-Raphson script:
- Automatically calculates the **symbolic derivative** using the `SymPy` library.
- Provides real-time iteration logs.
- Visualizes the function and the detected root.
